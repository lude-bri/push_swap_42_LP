

MANUAL TO USE VM on MAC

1. Install VM Box (Hypervisor)
2. Download ISO UBUNTU 22.04
3. Apply the ISO to VM Box
4. Install UBUNTU (at least 20GB)
5. VERIFY SSH
6. CONNECT VM TO PC

### also --> cgdb <name>

ACCESS ROOT

	(ALSO TRY THIS)
		1. usermod -a -G sudo vboxuser

	1. sudo nano /etc/sudoers
	2. write after "root ALL=(ALL:ALL) ALL" the following line "vboxuser ALL=(ALL:ALL) ALL"
	3. reboot VM
	4. sudo su

INSTALL SSH

	1. sudo apt update
	2. sudo apt install openssh-server
	3. sudo service ssh status

INSTALL VIM

	1. apt install vim

CONFIG SSH

	1. sudo nano /etc/ssh/sshd_config
	2. Change #Port 22 --> Port 22
	3. Change #PermitRootLogin prohibit-password -> PermitRootLogin no

	4. sudo nano /etc/ssh/ssh_config
	5. Change #Port 22 --> Port 22

	6. Restart --> sudo service ssh restart
	7. Check Status --> sudo service ssh status

INSTALL AND CONFIG FIREWALL

	1. sudo apt install ufw
	2. sudo ufw enable
	3. sudo ufw allow 22
	4. sudo ufw status


CONNECT VIA SSH

	1. In Virtual Box go to Configuration
	2. Network
	3. Advanced
	4. Post Forwarding
	5. Add Rule
	6. Add 22 to Host
	7. Add 22 to Guest
	8. Save changes
	9. Connect with --> ssh username@localhost -p 22
		Ex: ssh luigi@localhost -p 22


USING THE APT (PACKAGE MANAGER)

	1. Install curl ()
	2. Install GIT
	3. BREW (see the link online and follow the steps)
		3.1. Add brew to PATH
		3.2. Install BREW depedencies (build-essential)
		3.3. Instal GCC (brew install gcc)

INSTALL TMUX

	1. brew install tmux
	2. <leader> :set -g mouse

CONNECT GITHUB

	1. ssh-keygen
	2. cd .ssh/
	3. cat id_rsa.pub
	4. copy and add to github key
	5. ssh -vvv -T git@github
	6. git clone <repo>
